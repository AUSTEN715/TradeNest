# Zerodha Clone

A full-stack Zerodha-inspired stock trading platform clone built using the MERN stack. The project replicates core functionalities of a modern trading dashboard with responsive UI and dynamic data handling.

## Features

* User Authentication
* Dashboard Interface
* Holdings & Positions Management
* Responsive Trading UI
* REST API Integration
* Real-time Styled Financial Interface

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* Bootstrap/CSS
